ISO 9126
A ISO 9126 é uma norma criada para definir características que ajudam a avaliar a qualidade de um software.
As principais características são:

-Funcionalidade-
Verifica se o sistema realiza corretamente as funções para as quais foi criado.

-Confiabilidade-
Avalia se o software funciona de forma estável, sem falhas frequentes.

-Usabilidade-
Verifica se o sistema é fácil de aprender e utilizar.

-Eficiência-
Avalia o desempenho do sistema e o uso dos recursos disponíveis.

-Manutenibilidade-
Verifica a facilidade para corrigir erros e realizar melhorias no software.

-Portabilidade-
Avalia a facilidade de utilizar o sistema em diferentes ambientes e plataformas.

Essas características ajudam a medir a qualidade de um software e servem como referência para equipes de desenvolvimento e testes.
